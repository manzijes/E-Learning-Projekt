Kapitel 3

Verschiedene Medieneigenschaften

Jetzt geht es darum sogenannte Medieneigenschaften abzufragen. Das ist z.B. besonders nützlich für das Anpassen an Bildschirme. Natürlich zeige ich dir auch, wie der Code dazu aussehen kann.

Medieneigenschaften

Mit CSS3 wurde die Abfrage von Medieneigenschaften (Media Features) eingeführt und so das moderne responsive Design möglich gemacht.

Medieneigenschaften sind z.B.:

  • Größen von Viewports
  • Ausrichtungen von Viewports
  • Farbtiefen für Displays
  • Bildschirmauflösungen
  • Interaktionsfunktionen

Medieneigenschaften abfragen

Medieneigenschaften schreibst du in das media-Attribut des -Elements, die @import-Anweisung oder @media-Anweisung.

Das -Element verweist auf ein externes Stylesheet und setzt sich so zusammen. 

                                @media screen and (max-width: 480px) { 
                                    /* CSS-Anweisung */
                                }

Bei der @media-Anweisung sieht das ähnlich aus.

                                @import url("style.css") screen and (max-width: 480px);

Bei einer @import-Anweisung sieht die Syntax so aus.
