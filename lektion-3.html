Kapitel 3

Verschiedene Medieneigenschaften

Lernziele:

Du kannst, unabhängig von der CSS-Einbindungsart, verschiedene Medieneigenschaften abfragen und dadurch deine Seite u.a. an den Bildschirm der User anpassen.

Du kommst gut voran, weiter so!

Jetzt geht es darum, sogenannte Medieneigenschaften abzufragen. Das ist besonders nützlich für das Anpassen des Stylings an verschiedene Bildschirme. Natürlich zeige ich dir auch, wie der Code dazu aussehen kann.

Medieneigenschaften

Mit CSS3 wurde die Abfrage von Medieneigenschaften (Media Features) eingeführt und so das moderne responsive Design möglich gemacht.

Medieneigenschaften sind z.B.:

  • Größen von Viewports
  • Ausrichtungen von Viewports
  • Farbtiefen für Displays
  • Bildschirmauflösungen
  • Interaktionsfunktionen

Medieneigenschaften abfragen

Medieneigenschaften schreibst du in das media-Attribut des -Elements, die @import-Anweisung oder @media-Anweisung.

Das -Element verweist auf ein externes Stylesheet und setzt sich so zusammen. 

                                @media screen and (max-width: 480px) { 
                                    /* CSS-Anweisung */
                                }

Bei der @media-Anweisung sieht das ähnlich aus.

                                @import url("style.css") screen and (max-width: 480px);

Bei einer @import-Anweisung sieht die Syntax so aus.

Medieneigenschaften im Überblick

Wie zuvor in Kapitel 2 gebe ich dir hier eine Übersicht, diesmal über alle Medieneigenschaften, die mit CSS3 eingeführt wurden. Die meisten Medieneigenschaften sind mit den Präfixen min- und max- und zusätzlichen absoluten oder relativen Größen verwendbar.

Media Feature Beschreibung Werte
width (min-/max-) Breite der sichtbaren Ausgabe (Viewport) px, %, em
height (min-/max-) Höhe der sichtbaren Ausgabe (Viewport) px, %, em
device-width Breite des Displays des Ausgabegeräts (oder Breite einer gedruckten Seite) px, %, em
device-height Höhe des Displays des Ausgabegeräts (oder Höhe einer gedruckten Seite) px, %, em
orientation Prüft die Ausrichtung eines Ausgabegeräts. portrait, landscape
aspect-ratio (min-/max-) Prüft das Seitenverhältnis des Ausgabegeräts Breite/Höhe, z.B. 16/9
device-aspect-ratio (min-/max-) Prüft das Seitenverhältnis des Ausgabegeräts Breite/Höhe, z.B. 16/9
color Prüft die Farbtiefe des Ausgabegeräts integer
color-index (min-/max-) Prüft das Ausgabegerät auf die Verwendung von indizierten Farben integer
monochrome (min-/max-) Prüft die Farbtiefe auf monochromen Ausgabegeräten. monochrome:0 wäre der Wert für nicht monochrome Geräte integer
resolution Prüft die Pixeldichte in dpi (dots per inch) oder dpcm (dots per centimeter) dpi, dpcm
scan Testet TV-Geräte auf die Bilddarstellungsarten Vollbild (progressive) und Halbbild (interlace) progressive, interlace
grid Prüft die Rastereigenschaften. Bei Ausgabemedien, die Inhalte in einem Raster darstellen, ist der Wert 1, bei einer Bitmap-Darstellung ist der Wert 0 0, 1
pointer, any-pointer Prüft, ob das Ausgabegerät eine Maus als primäres Eingabegerät oder überhaupt ein Eingabegerät zur Verfügung stellt none, coarse, fine
hover, any-hover Prüft, ob das Ausgabegerät Hovereffekte beim primären Eingabegerät oder überhaupt zur Verfügung stellt none, hover
